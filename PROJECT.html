MY PROJECT

Hey there! I’m working on a project that involves creating a parallax scrolling website.
This type of website uses the parallax effect to create a 3D-like effect by moving the
background of a web page at a different speed than the foreground. This can make the
website feel more interactive and engaging. However, it’s important to note that
excessive movement within parallax effects can cause harm for those with vestibular
disorders. To create a parallax scrolling website, I’m using HTML, CSS, and JavaScript.
There are many online resources available that can help you get started with this.
Good luck with your website!”
